### 如何更新相册图片

本站是静态站点，没有网页内直接上传图片的后台。要新增图片，需要修改组件里的数据后重新构建部署。

- 上方九宫格：编辑 `docs/.vuepress/components/ProjectNineGrid.vue` 里的 `items`
- 下方分组相册：编辑 `docs/.vuepress/components/AlbumFeed.vue` 里的 `blocks` 与 `urls`

## 足迹地图

### 中国大陆
- 北京：故宫、颐和园、长城
- 上海：外滩、豫园、朱家角
- 成都：宽窄巷子、熊猫基地、川西
- 西安：兵马俑、城墙、回民街

### 中国香港
- 中西区：太平山顶、荷里活道
- 九龙：旺角、尖沙咀海滨长廊
- 离岛：大屿山、南丫岛
- 新界：大埔墟、沙田

### 亚洲其他
- 新加坡：滨海湾、圣淘沙、小印度
- 东京：新宿、浅草、台场
