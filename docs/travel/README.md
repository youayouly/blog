## 旅行小贴士

> 💡 **香港交通**：用八达通卡，地铁+巴士全覆盖，比内地线路复杂但效率高。

> 💡 **新加坡住宿**：宿舍外住宿推荐 Little India 或 Chinatown 附近，性价比高。
