# 🎓 留学规划

整理我在规划留学过程中收集的资讯与经验，覆盖香港、英国、新加坡三个热门留学目的地。

---

## 三地概览

| 目的地 | 特点 | 适合人群 |
|--------|------|----------|
| 🇭🇰 香港 | 近内地、国际化、性价比高 | 想留在大湾区发展 |
| 🇬🇧 英国 | 历史名校、PSW 签证政策好 | 想获英国永居/工作经验 |
| 🇸🇬 新加坡 | 双语环境、亚洲金融中心 | 想进入东南亚职场 |

## 通用申请时间线

```
10 月前   → 确定目标院校，准备文书材料
10–12 月  → 提交申请（英国 UCAS / 港申请系统）
1–3 月    → 等待 Offer，准备语言考试
4–5 月    → 确认录取，办理签证
7–8 月    → 准备出发，安排住宿
```

> 具体时间因学校和项目而异，请以官方公告为准。
