# 留学规划

这里整理我在准备留学过程中收集的信息、比较逻辑和实际判断，覆盖中国香港、英国和新加坡三个主要方向。

---

## 三地总览

| 地点 | 特点 | 适合方向 |
|------|------|----------|
| 中国香港 | 距离近、国际化程度高、节奏快 | 想连接大湾区与国际机会 |
| 英国 | 学校资源集中、项目成熟、申请体系清晰 | 想积累海外学习与工作经历 |
| 新加坡 | 双语环境、亚洲枢纽、生活效率高 | 想进入东南亚与国际化职场 |

## 通用申请时间线

```text
10 月前    -> 明确目标院校，准备文书与材料
10-12 月   -> 提交申请
1-3 月     -> 等待 offer，补充语言成绩与材料
4-6 月     -> 确认录取，处理签证与住宿
7-8 月     -> 准备出发与行前安排
```

> 具体节奏会因学校和项目而不同，请以官方通知为准。
